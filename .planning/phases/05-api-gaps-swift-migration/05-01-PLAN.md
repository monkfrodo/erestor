Create three new FastAPI router files following the exact patterns from existing routers (status.py, context.py). All three use `router = APIRouter(dependencies=[Depends(verify_token)])` and return `ApiResponse`.

**1. `api/routers/timer.py` -- POST /timer/stop**

Stops whatever timer is currently active. Logic:
- Check each timer file in order: `$HOME/.work_timer`, `$HOME/.content_timer`, `$HOME/.ocio_timer`, `$HOME/.study_timer`
- For the first one that exists: read start_ts, compute duration, delete the timer + desc files
- Save to GCal using `gcal_extend_or_create` (import from `erestor.calendar`) with the correct calendar ID per type (import from `erestor.config`: GCAL_TRABALHO, GCAL_CONTEUDO, GCAL_TECNOLOGIA, GCAL_ESTUDOS)
- Emit `timer.stopped` event via `erestor.event_bus.emit`
- Return ApiResponse with data: `{"stopped": true, "type": "work|content|tech|study", "duration_mins": N, "desc": "..."}`
- If no timer is active: return ApiResponse with `data={"stopped": false, "message": "Nenhum timer ativo"}`
- Use `asyncio.to_thread()` for all sync calls (GCal, file I/O) to avoid blocking
- Use `pathlib.Path.home()` for HOME (not hardcoded path)
- Import `BRT` from `erestor.config` for timezone-aware datetime formatting

**2. `api/routers/history.py` -- GET /history**

Returns recent conversation history. Query params: `source` (optional, "desktop"|"telegram"), `limit` (optional int, default 10, max 50).
- Import and call `load_history()` from `erestor.memory` via `asyncio.to_thread()`
- Filter by source if provided
- Slice to limit
- Return ApiResponse with `data={"history": [...]}` where each entry is `{"u": "...", "b": "..."}`

**3. `api/routers/device.py` -- POST /device/register**

Stores iOS device tokens for APNs push.
- Accept JSON body: `{"token": str, "platform": "ios"}` (use Pydantic model `DeviceRegisterRequest`)
- Read existing `~/claude-sync/produtividade/erestor/data/devices.json` (create if missing, create `data/` dir if missing)
- Check if token already registered (avoid duplicates)
- Append `{"token": token, "registered_at": datetime.utcnow().isoformat()}` to the `ios` array
- Write back atomically
- Return ApiResponse with `data={"registered": true}`
