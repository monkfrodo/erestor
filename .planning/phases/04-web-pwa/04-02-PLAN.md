  <name>Task 1: Chat tab with streaming markdown responses</name>
  <files>
    web/src/components/tabs/ChatTab.tsx, web/src/components/chat/ChatMessage.tsx,
    web/src/components/chat/ChatInput.tsx, web/src/services/chat.ts,
    web/src/__tests__/chat.test.tsx

  <action>
    1. Create `web/src/services/chat.ts`:
       - Export `streamChat(message: string)` function
       - POST to `${API_BASE}/v1/chat/stream` with Bearer token, JSON body `{ message, history }` where history = last 20 messages from chatStore
       - Read response via `response.body.getReader()` + TextDecoder
       - Parse SSE format: split on `\n`, extract `data:` lines, JSON.parse each
       - On `data.text`: call `useChatStore.getState().appendToken(data.text)`
       - On `data.done`: call `useChatStore.getState().finishStreaming(data.full_response, data.actions)`
       - Handle errors: if fetch fails or stream errors, add error message to chat store
       - Do NOT use EventSource (POST not supported by EventSource API)

    2. Create `web/src/components/chat/ChatMessage.tsx`:
       - Props: message object with role (user/assistant), content, isStreaming flag
       - User messages: right-aligned, DS.s2 background, DS.bright text
       - Assistant messages: left-aligned, transparent background, DS.text color
       - When isStreaming=true: render content as plain text with blinking cursor indicator
       - When isStreaming=false and role=assistant: render via react-markdown with remark-gfm and rehype-highlight
       - Apply Vesper Dark theme to code blocks (background var(--ds-s2), text var(--ds-bright))
       - Import highlight.js dark theme CSS (or custom CSS matching Vesper palette)

    3. Create `web/src/components/chat/ChatInput.tsx`:
       - Fixed-position input at bottom of chat container
       - Text input + send button
       - DS styling: bg var(--ds-s2), border var(--ds-border), text var(--ds-bright), placeholder var(--ds-subtle)
       - Send on Enter (not Shift+Enter which is newline) and on send button click
       - Disabled state when isStreaming=true (from chatStore)
       - Props: onSend(message: string) callback

    4. Create `web/src/components/tabs/ChatTab.tsx`:
       - Flex column: scrollable message list (flex-1) + ChatInput at bottom
       - Auto-scroll to bottom on new messages and during streaming
       - useRef on message container, scrollIntoView on message changes
       - On send: add user message to chatStore, call streamChat(message), chatStore adds assistant placeholder
       - Map messages from useChatStore to ChatMessage components
       - Empty state: centered "Converse com Erestor" message

    5. Create `web/src/__tests__/chat.test.tsx`:
       - Test ChatMessage renders user message right-aligned
       - Test ChatMessage renders assistant message with markdown (mock react-markdown)
       - Test ChatInput calls onSend on Enter key
       - Test ChatInput is disabled when isStreaming=true
       - Test streamChat parses SSE data correctly (mock fetch with ReadableStream)
