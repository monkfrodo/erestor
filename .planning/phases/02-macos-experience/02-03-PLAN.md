       - Parse SSE lines: look for "data: " prefix, decode JSON

         - `gate_alert` -> append to new `@Published var activeGates: [GateSSEEvent]`

         - `heartbeat` -> update `self.serverOnline = true`, reset failure counter
       - On error/disconnect: wait 3s, reconnect (exponential backoff up to 30s)

       - On "done" event: set `isStreaming = false`, parse actions
    5. Update `init()` to call `startEventStream()` instead of polling methods
    6. Keep `serverOnline` logic but driven by heartbeat timeout (no heartbeat for 60s = offline)
